Execution du programme

Une fois le fichier dé-zippé, insérer votre modèle Yolo dans le dossier et démarrer l'application avec la ligne de commande -> streamlit run app.py. Une fois dans l'application, utilisez le téléverseur de fichier pour ajouter une image et démarrer la détection.


Questions de réflexions

1- Lorsqu'une image ne contenant pas de voiture est chargée, un message disant qu'aucune voiture n'est détectée apparaît au lieu du tableau de résultats brut.

2- Les scores de confiances sont affichés sur l'image après la détection. Pour les voitures, le cadrage est blanc, rouge pour les personnes, jaune pour les moto et cyan pour les camionnettes.

3- Nous avons optés pour ajouter une barre de navigation latérale.

4- Notre modèle détecte rapidement les images données.
